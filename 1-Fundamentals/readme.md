ストラングの書籍を読んでいたために、序盤の行列の基本演算や、列ベクトル単位での見方といった内容はサクサク読むことができました。ストラングの本は線形代数学の基本的な内容から教えているため、
複素数の範囲は応用的な内容まで進めてもそこまで深く触れられていませんでした。共役転置や、エルミート行列は新しい知識が比較的多かったです。また、内積は転置の積だと思っていたので、共役転置まで拡大された説明で知識を深めることができました。

ノルムについてです。ストラングでは「条件数」として記載され、行列Aによってどれだけxの誤差が拡大するかの相対評価としてAx/xが出てきました。「ホルダーの不等式」や「列ベクトルの和の最大値」で判別できるといった内容は、より現実的で
有限時間での判別を視野に入れているような内容だと感じました。


フロベニウスノルムに関しては本当にそうなのか気になったので、python3でコードを組んで実験してみました。説明通り一致はするのはもちろんですが、素直に「すげぇ」という気持ちになりました。
知識として理解するだけでなく、実際に自分で検証して「本当にそうなる」と確かめた経験こそが、後で活用するときの自信につながると思っています。検証できるものは今後も積極的に行っていきたいと考えています。
