# Numerical Linear Algebra Notes

『NUMERICAL LINEAR ALGEBRA』

## 目的
- 書籍内容の整理と理解メモ
- 読書過程そのものの記録（取り組み日記）

## 方針
- 各セクションで読んで理解ができなかった点を調べ、自分用に補足する
- 教科書と並行して参照できる補助ノートとしてまとめる
- 理解した内容は消さずに、追記・更新する

## 補足
- 内容の完成度よりも理解の過程を優先している
- 他者向けの内容ではない
- 数式環境がLuaLaTeXと一致した出力とならなかったため、pdfを投稿して更新
- pdf名は投稿されたpdfの記載内容までを学習し終えた時の日付であり、TeXの作成日ではない。
