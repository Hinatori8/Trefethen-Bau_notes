## 目的
- 書籍内容の整理と理解

Strangの本のQR分解からがとても面白く、大学で履修した「数値解析」が面白かったので、この本を読むことにしました。この本はQR分解、SVD、Gram-Schemidtだけでなく、反復法も解説しているので、自分の興味に合う気がします。
## 方針
- 読んだ内容を自分の言葉に変換して丁寧に読んでいこうと思います。
