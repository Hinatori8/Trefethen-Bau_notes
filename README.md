## 目的
- 書籍内容の整理と理解メモ
- 読書過程そのものの記録（取り組み日記）

## 方針
- 各セクションで読んで理解ができなかった点を調べ、自分用に補足する
- 教科書と並行して参照できる補助ノートとしてまとめる
- 理解した内容は消さずに、追記・更新する

Gilbert Strang 著の和訳版『STRANG:線形代数イントロダクション』を主要教材として学習していた。第1-6,8,9章は納得するまで繰り返し読み、他の章はサッと目を通した程度です。

## 補足
- 理解の過程を優先している。
- 他者向けの内容ではない。
- 各セクション内の「read.md」はその章に関する私の考えや感想、伝えたいことを自由に記載している。
- 数式環境がLuaLaTeXと一致した出力とならなかったため、pdfを投稿することにしている。
- 英語(書籍の文章)のままの方が理解できる箇所は、英文のまま記載。
- pdf名は投稿されたpdfの記載内容までを学習し終えた時の日付であり、TeXの作成日ではない。
- 投稿済み内容で補足された場合は、投稿済みのpdfは削除せず、追加する形で過去の記録も残している。
